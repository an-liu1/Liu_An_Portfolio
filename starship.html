    STARSHIP TROOPERS

Overview

This is a final project in my second semester in Fanshawe College, which is a responsive website about the movie --- STARSHIP TROOPERS. It includes JS movies' controls, AE trailers, and a JS game, which represents the most works I learnt in this semester. The task was to create a responsive website to fully show all the information abnout this movie --- Starship Troopers in order to promote the movie, and there is a contact section to get feedback from viewers. I was asked to do the basic design for this website, create a movie trailer for this movie by AE, and work on javascript functions to do some animations(lightbox, video controls, and slider).

Starship Troopers Trailer

A trailer is an advertisement or a commercial for a feature film that will be exhibited in the future at a cinema, the result of creative and technical work.

Brief

When I started with this project, I was thinking about how to make this site simple and can fully show all the information to viewers. The game panel is the the key breakthrough point. The main plot in this movie is all soldiers shoot bugs in order win the war. The whole movie is full of passion, so I picked red as the main color in this site and put the shooting game I made in the first section to let viewers get this kind of feeling. I also put four dynamic bugs on the site to produce the scenario. I chose the color --- white as the background color, which means the hope in the war.

Brand Identity

Starship Troopers’ brand identity system provides strength to the movie by the special font and size. The original design of this logo looks almost like this with only color change. The main plot in this movie is about the war in space, so I chose the red as the main color in this website to show more passion and fighting feeling to viewers. The red color of this logo is for the website of full red system. Both S and T are the capital letter to show obviously to audiences the main ideas of this story happened in the universe.
